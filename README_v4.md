## 🚀 Установка и запуск

### Шаг 1: Обновить requirements.txt

```
python-telegram-bot==20.7
python-dotenv==1.0.0
groq==0.9.0
matplotlib==3.7.2
pillow==10.0.1
aiohttp==3.9.0
```

### Шаг 2: Отредактировать `.env`

```bash
TELEGRAM_BOT_TOKEN=your_token_here
GROQ_API_KEY=your_groq_key_here
```

**Как получить ключи:**

1. **Telegram Bot Token** - напиши `/newbot` в @BotFather в Telegram
2. **Groq API Key** - регистрация на https://console.groq.com (БЕСПЛАТНО!)

### Шаг 3: Установить зависимости

```bash
pip install -r requirements.txt
```

### Шаг 4: Запустить основной бот

```bash
python bot_v4_with_ai.py
```

### Шаг 5 (ОПЦИОНАЛЬНО): Запустить напоминания в отдельном терминале

```bash
# Тестовый режим (проверит сейчас)
python reminder_scheduler.py test

# Продакшн (будет проверять каждый день в 9:00 UTC)
python reminder_scheduler.py
```

## 📱 Основные функции

### 💪 Категории упражнений

1. **Грудь и бицепс** - Жим, сведения, подъёмы
2. **Спина и трицепс** - Тяги, отжимания
3. **Ноги и плечи** - Приседания, жим ногами, махи

**Действия для каждого упражнения:**
- ✏️ Изменить вес (+ автосохранение в историю)
- 📈 История весов с датами
- 📊 График прогресса
- 💬 AI совет по технике
- ❌ Удалить упражнение

---

### 🤖 AI Тренер в личном чате

**Доступные опции:**

1. 💪 **Мотивация** - Короткая мотивирующая фраза
2. 🎯 **Советы по технике** - Правильное выполнение
3. 🍗 **Совет по питанию** - Восстановление и еда
4. ❓ **Свой вопрос** - Спроси что угодно

**Особенности:**
- Context-aware (знает про твоего тренировочный стиль)
- Персональный стиль: "малыш", "сука" 😎
- Fallback ответы если API недоступен
- Автоматическое кэширование ответов

## 🐛 Отладка

### Проблема: "GROQ_API_KEY not found"
**Решение:** Проверь `.env` файл, должна быть строка:
```
GROQ_API_KEY=gsk_XXXXXXXXXXXXX
```

### Проблема: "No module named 'groq'"
**Решение:**
```bash
pip install groq aiohttp
```

### Проблема: AI не отвечает
**Решение:** 
- Проверь интернет
- Проверь API ключ на console.groq.com
- Посмотри логи (bot выпишет ошибку)

### Проблема: БД не создаётся
**Решение:** БД автоматически создаётся при первом запуске. Если ошибка:
```python
from database_with_ai import create_tables
create_tables()  # Запусти это вручную
```

## 🚀 Развёртывание на Railway/Heroku

Для облачного деплоя:

1. Создай `Procfile`:
   ```
   web: python bot.py
   worker: python reminder_scheduler.py
   ```

2. Добавь в Railway/Heroku переменные:
   ```
   TELEGRAM_BOT_TOKEN=xxx
   GROQ_API_KEY=xxx
   ```

3. Deploy:
   ```bash
   git push heroku main
   ```
