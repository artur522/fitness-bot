## 📋 Инструкция по запуску

### Вариант 1: Быстрый старт (5 мин)

```bash
cd worker_git

# 1. Обновить зависимости
pip install -r requirements.txt

# 2. Убедиться что .env есть
cat .env  # должно быть:
# TELEGRAM_BOT_TOKEN=xxx
# GROQ_API_KEY=xxx

# 3. Запустить бот
python bot_v4_with_ai.py
```

### Вариант 2: Полная интеграция (с заменой)

```bash
cd worker_git

# Backup старых
mv database.py database_old.py
mv bot.py bot_old.py

# Новые вместо старых
mv database_with_ai.py database.py
mv bot_v4_with_ai.py bot.py

# Запустить
python bot.py
```


## 🚀 Deployment

### На Railway.app

1. Создай `Procfile`:
```
web: python bot_v4_with_ai.py
worker: python reminder_scheduler.py
```

2. Добавь env vars на Railway
3. Deploy

### На Heroku

```bash
heroku create your-app-name
heroku config:set TELEGRAM_BOT_TOKEN=xxx
heroku config:set GROQ_API_KEY=xxx
git push heroku main
```

### На VPS/Server

```bash
nohup python bot_v4_with_ai.py > bot.log 2>&1 &
nohup python reminder_scheduler.py > reminders.log 2>&1 &
```
