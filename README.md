# 🤖 Фитнес-Бот v4.0 с AI

Умный бот для отслеживания прогресса в тренировках с AI тренером.

## 🚀 Быстрый старт

```bash
# Установить зависимости
pip install -r requirements.txt

# .env должен иметь:
TELEGRAM_BOT_TOKEN=your_token_here
GROQ_API_KEY=your_groq_key_here

# Запустить бот
python bot_v4_with_ai.py

# (опционально) Напоминания в отдельном терминале
python reminder_scheduler.py
```

## ✨ Фичи

- 💪 Управление упражнениями по категориям
- 📊 Графики прогресса
- 🤖 AI Тренер (мотивация, советы по технике, питание)
- 🔔 Автоматические напоминания (5+ дней неактивности)
- 📈 История весов с датами
- 🎨 Красивый интерфейс с эмодзи

## 📁 Структура

```
├── bot_v4_with_ai.py       # Основной бот
├── database_with_ai.py      # БД с user_stats
├── reminder_scheduler.py    # Автоматические напоминания
├── requirements.txt         # Зависимости
└── README.md                # Этот файл
```

## 🔧 Требования

- Python 3.8+
- Telegram Bot Token (от @BotFather)
- Groq API Key (бесплатно на console.groq.com)

## 🎯 Использование

1. Напиши боту `/start`
2. Выбери категорию упражнений
3. Управляй своими упражнениями
4. Получай советы от AI тренера
5. Смотри графики прогресса

## 📝 Примеры

**AI Консультации:**
- 💪 Мотивация - эмоциональный boost
